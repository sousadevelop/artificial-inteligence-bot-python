# Merlin AI

Educational Windows virtual assistant with speech recognition, text-to-speech, and local shortcuts.

**Category:** Desktop Assistant

**Stack:** Python, pyttsx3, SpeechRecognition, keyboard, python-decouple

**Status:** Portfolio-ready educational project

## Run Locally

```bash
python main.py
```
